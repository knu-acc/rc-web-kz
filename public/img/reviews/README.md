# Папка для изображений отзывов

В эту папку необходимо добавить скриншоты реальных отзывов клиентов.

## Требования к изображениям:
- Формат: PNG или JPEG/JPG
- Именование: review-1.png, review-2.png и т.д.
- Рекомендуемое соотношение сторон: 3:4 (вертикальное)
- Рекомендуемый размер: 400x600 пикселей

## После добавления:
Обновите файл `data/reviews.ts` с путями к новым изображениям.
