# Чек-лист перед финальной сборкой

## Перед сборкой

- [ ] Все изображения загружены в `public/img/`
- [ ] Проверены контакты в `lib/constants.ts` (телефон, email, адрес)
- [ ] Проверен URL сайта в `lib/constants.ts` (должен быть `https://rc-web.kz`)
- [ ] Удалены все временные файлы и тестовые данные
- [ ] Проверены все ссылки на сайте

## Сборка

```bash
# 1. Очистка предыдущих сборок (опционально)
npm run clean

# 2. Установка зависимостей (если нужно)
npm install

# 3. Сборка проекта
npm run build
```

## После сборки

- [ ] Проверьте папку `out/` - все файлы должны быть там
- [ ] Скопируйте `public/.htaccess` в `out/.htaccess`
- [ ] Проверьте наличие `sitemap.xml` и `robots.txt` в `out/`
- [ ] Откройте `out/index.html` в браузере и проверьте базовую работу
- [ ] Проверьте консоль браузера на ошибки (F12)

## Перед загрузкой на хостинг

- [ ] Убедитесь, что все изображения в `out/img/` на месте
- [ ] Проверьте размер папки `out/` (не должна быть слишком большой)
- [ ] Создайте резервную копию старого сайта (если есть)

## После загрузки на хостинг

- [ ] Откройте сайт в браузере
- [ ] Проверьте работу всех страниц
- [ ] Проверьте редиректы (старые URL должны перенаправлять)
- [ ] Проверьте `robots.txt`: `https://ваш-сайт.ru/robots.txt`
- [ ] Проверьте `sitemap.xml`: `https://ваш-сайт.ru/sitemap.xml`
- [ ] Проверьте мета-теги через инструменты:
  - [Google Rich Results Test](https://search.google.com/test/rich-results)
  - [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)
- [ ] Проверьте работу форм обратной связи
- [ ] Проверьте мобильную версию

## SEO проверка

- [ ] Все страницы имеют уникальные title и description
- [ ] Все изображения имеют alt атрибуты
- [ ] Schema.org разметка присутствует (проверьте через Google Rich Results Test)
- [ ] Внутренняя перелинковка работает
- [ ] Нет битых ссылок (404 ошибок)

## Производительность

- [ ] Страницы загружаются быстро (< 3 секунд)
- [ ] Изображения оптимизированы
- [ ] Нет ошибок в консоли браузера
- [ ] Мобильная версия работает корректно
